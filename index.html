PICamRU

I worked with my team to make a dashcam using a Raspberry Pi for our Senior Capstone Design project

The goal of this project was to reduce the amount of potentially harmful driving habits that would occur on the road. The back-end utilizes Google's sample TensorFlow object detection model with other Python libraries to identify these driving habits.

Drip 'n' Dash

I worked with my team to make an Android Application using Java for our Software Engineering Semester Project

This concept was made in order for registered students to earn money while completing menial chores for other students. Utilizing Google's Firebase allowed for a safe connection between the client and the "Dasher" to exist. Progress bars are present to show the status of the job, from the point where the job is accepted to the point of completion.
